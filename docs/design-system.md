# Design System — SimpliPlan

> Quelle: `public/SimpliPlan PRD.rtf`, Abschnitt "CI-Richtlinien". Diese Werte sind verbindlich für alle UI-Arbeiten (`/frontend`).

## Farben

Die Farbpalette basiert auf Blautönen mit Gelb für Gemeinschaft, kombiniert mit neutralen Tönen für ein modernes, klares Design.

| Rolle | Name | HEX | Verwendung |
|---|---|---|---|
| Primär | SimpliPlan Blau | `#00335E` | Buttons, Überschriften, Header, Akzente |
| Akzent | Dunkelgelb | `#E9BF08` | Primäre CTA-Buttons (z.B. Registrieren), Hover-States, sekundäre Buttons |
| Hintergrund | Weiß | `#FFFFFF` | Haupthintergrund |
| Hintergrund sekundär | Hellgrau | `#F5F5F5` | Sekundärhintergründe, Trennbereiche |
| Text | Dunkelgrau | `#333333` | Textfarbe |

## Typografie

- **Überschriften:** Schriftfamilie **Rubik** (modern, gut lesbar, Sans-Serif)
- **Kleinere Überschriften, Fließtext, Sekundärtext:** Schriftfamilie **Raleway**

### Schriftgrößen
| Element | Größe | Gewicht |
|---|---|---|
| Hauptüberschriften | 1.5rem (24px) | bold |
| Unterüberschriften | 1.25rem (20px) | bold |
| Abschnittsüberschriften | 1rem (16px) | semibold |
| Kleinere Überschriften | 0.875rem (14px) | semibold |
| Fließtext | 0.875rem (14px) | regular |
| Sekundärtext | 0.875rem (14px) | regular |

Wichtige Schlüsselbegriffe können **fett** oder in **SimpliPlan Blau** hervorgehoben werden.
